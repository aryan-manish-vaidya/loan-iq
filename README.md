# 🚀 Loan IQ

> Intelligent Loan Eligibility & Risk Assessment Platform

Loan IQ is a full-stack financial technology application designed to streamline loan evaluation and eligibility analysis. The platform enables users to assess loan applications, analyze risk factors, and make informed lending decisions through an intuitive web interface and a powerful Python backend.

---

## 📖 Overview

Financial institutions and lenders often face challenges in evaluating loan applications efficiently and consistently. Loan IQ addresses this problem by providing a centralized platform that assists in analyzing loan data, assessing eligibility criteria, and supporting decision-making processes.

The application combines a modern React frontend with a Python-powered backend to deliver a fast, scalable, and user-friendly experience.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/aryan-manish-vaidya/loan-iq.git
cd loan-iq
```

### 2. Install Backend Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 🚀 Running the Application

### Backend

```bash
cd backend
python app.py
```

### Frontend

```bash
cd frontend
npm start
```

### Using Shell Script

```bash
chmod +x start.sh
./start.sh
```
